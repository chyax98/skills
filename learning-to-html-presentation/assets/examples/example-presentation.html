什么是 Hooks?

  • React 16.8 引入的新特性
  • 让函数组件可以使用 state 和其他 React 特性
  • 无需编写 class 组件

useState Hook

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      点击次数: {count}
    </button>
  );
}
